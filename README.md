## 🧠 About Me
**Building the infrastructure that makes AI systems reliable, fast, and production-ready.**

I'm an AI Backend Engineer specializing in the design and scaling of retrieval-augmented generation (RAG) pipelines, knowledge graphs, and LLM inference systems. I care deeply about performance, architecture, and shipping AI that actually works at scale.

- 🔭 **Currently building:** Scalable knowledge retrieval pipelines using GraphRAG and vector-native architectures
- 🤝 **Open to collaborating on:** Full-stack AI systems, GenAI experimentation, and production RAG implementations
- 🌱 **Currently exploring:** Embedding optimization, RLMS, Titans architecture, and real-time AI serving
- ⚡ **Fun fact:** I weave current affairs and history into my AI projects wherever I can — context is everything

## 🚀 What I'm Working On
| Area | Focus |
|------|-------|
| **Retrieval Systems** | Designing GraphRAG pipelines that go beyond naive chunk retrieval |
| **LLM Inference** | Reducing latency and cost in production LLM deployments |
| **Embedding Pipelines** | Optimizing dense retrieval with custom embedding strategies |
| **AI APIs** | Shipping clean, documented FastAPI services backed by vector databases |
